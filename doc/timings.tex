\begin{itemize}
\item \texttt{CCD O} to \texttt{In 2} as handshake for ``CCD ready'', start synchronous
  triggering, one or two scans can be dropped
\item
\texttt{TC0} triggered on external event \texttt{TCLK0} (\texttt{laser in}), retriggered
\item Count output pulses on \texttt{TIOA0}, stop after scans per block
\item
\texttt{TC1} triggered on external event \texttt{TCLK0}, enabled on every other external
  event trigger, not retriggered
\item debug laser trigger: \texttt{TC6} free running clock on
  \texttt{TIOA6} / \texttt{Out 4}
\item debug fs PD: \texttt{TC2} triggered on external event \texttt{TCLK0},
  \texttt{TIOA2} / \texttt{Out 3}
\item debug ps PD: \texttt{TC8} triggered on external event \texttt{TIOA6} rising slope
  \texttt{TIOA2} / \texttt{Out 3}, no retrigger, steered by main trigger interrupt
\end{itemize}
